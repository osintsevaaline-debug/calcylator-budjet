# Как открыть проект на компьютере

Проект находится в папке `Задание 1` и состоит из:

- `index.html` - приветственный лендинг
- `калькулятор расходов.html` - основной функционал калькулятора
- `styles.css` - стили для лендинга

## Способ 1 (самый простой)

1. Откройте папку `c:\Users\admin\Desktop\курсор\Задание 1`.
2. Найдите файл `index.html`.
3. Дважды кликните по `index.html`.
4. Откроется лендинг, затем нажмите кнопку **Открыть калькулятор**.

## Способ 2 (через Live Server в Cursor/VS Code)

1. Откройте папку проекта в Cursor.
2. Установите расширение **Live Server** (если еще не установлено).
3. Нажмите правой кнопкой по `index.html`.
4. Выберите **Open with Live Server**.
5. Сайт откроется в браузере, обычно по адресу `http://127.0.0.1:5500/`.

## Если стили лендинга не применились

- Проверьте, что `index.html` и `styles.css` лежат в одной папке.
- Убедитесь, что в `index.html` есть строка:

```html
<link rel="stylesheet" href="styles.css">
```

## Как быстро обновить страницу после изменений

- Нажмите `Ctrl + F5` в браузере, чтобы сбросить кеш и увидеть актуальную версию сайта.
