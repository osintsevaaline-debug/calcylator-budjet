# Family Budget App

Небольшой учебный веб-проект на чистом HTML/CSS/JS для учета семейного бюджета.

## Что есть в проекте

- Приветственный лендинг: `index.html`
- Рабочий калькулятор бюджета: `калькулятор расходов.html`
- Стили лендинга: `styles.css`

## Функционал калькулятора

- Подсчет доходов, обязательных и переменных расходов
- Расчет остатка бюджета и процента накоплений
- Финансовые рекомендации по результатам
- Постановка месячной цели и отображение прогресса
- Сохранение данных в `localStorage`
- Экспорт/импорт данных в формате JSON
- Снимки бюджета по месяцам (история)
- Переключение светлой и темной темы

## Быстрый запуск

1. Откройте папку проекта.
2. Запустите `index.html` двойным кликом или через Live Server.
3. На лендинге нажмите **Открыть калькулятор**.

## Структура

```text
Задание 1/
```
